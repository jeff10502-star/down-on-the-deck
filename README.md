# Down on the Deck

A crop-duster arcade game — fly low, keep the spray on, clear each field. Built as a single-file HTML5 game / installable PWA.

**Play:** https://jeff10502-star.github.io/down-on-the-deck/

Hold to climb, release to sink. Spray is heaviest on the deck. Pull up over fences, trees, and power lines; duck under or climb over the tall highlines; dodge cattle, deer, and the odd bird. Grab tokens for bonus points and power-ups (shield, spray boost, 2x score). Clear the acre goal each field — longer unbroken swaths multiply your score. Three lives.
